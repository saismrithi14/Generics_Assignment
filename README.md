The purpose of this repository is to practice questions in Java using Generics

Below are the questions for the assignment:

Q1.Create a generic class Box<T> that:

Stores a single value of type T that has

    1) void set(T value)

    2) T get()

    3) A single value

Override toString() to print the value 

Q2.Write a generic static method that finds the maximum of 3 objects

Requirements:
    
    1) Returns the largest of 3 values
 
    2) Must work with:
        
        a) Integers

        b) Double

        c) String

        d) Any custom class impelmenting Comparable

Q3. Create a generic custom stack class


Methods that the stack should implement
    
    1) Push

    2) pop

    3) peek

    4) isEmpty

Q4. Implement a generic method that copies elements from one list to another.
